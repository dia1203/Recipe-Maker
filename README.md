# Recipe Maker

A web application that helps users create, view, and manage recipes through a simple and user-friendly interface.

## Technologies Used
- HTML
- CSS
- JavaScript

## Features
- Add and manage recipes
- View recipe details
- Recipe rating system
- Display cooking/preparation time
- User-friendly interface
- Responsive design

## Objective
To provide an easy platform for users to organize recipes, rate them, and check the estimated preparation time.
